# Pika-Pika
Live a cute pika 귀여운 쥐토끼의 삶을 살아봅시다 

## Description

- 등 따숩게 대충 누워서 쥐토끼생 살고 싶은데, 열심히 독초를 채집하지 않으면 HP가 열심히 깎이므로 주의합시다.
- 다른 쥐토끼하고 같이 열심히 독초 채집해서 살림을 일구거나 동료의 살림살이를 열심히 훔쳐봅시다.
- 쥐토끼마다 행동 패턴이 다를 건데 알아서 잘 파악해봅시다.

## 등장 쥐토끼
- Pika: 그냥 쥐토끼
- Rpika: 알파카처럼 털이 복슬복슬한 쥐토끼
- Pyka: 매끈매끈한 쥐토끼
- Hika: 카우보이 쥐토끼

## 규칙

- 타이틀 화면에서 게임시작 버튼 누르고 시작합시다.
- 본격적인 게임시작 전 플레이 하고 싶은 쥐토끼를 선택합시다.
- 쥐토끼를 선택하고 간단한 스토리를 텍스트로 보여줍니다. 플레이어가 원하면 다음 장면으로 바로 넘어갈 수 있게 SKIP 버튼도 만들어줍시다.
- 스토리가 끝나면 규칙 설명 및 조작방법 화면과 함께 게임시작 버튼이 나타납니다. 버튼을 눌러봅시다. 그러면 본 게임이 5초 카운트 후 시작합니다.
- 사각형맵 각 모서리에 각 쥐토끼의 굴이 있습니다. 쥐토끼 굴 위치는 매 게임마다 랜덤하게 배정됩니다.
- 가장 먼저 자기 굴에 풀무더기를 30개 쌓은 쥐토끼가 승리합니다.
- 사각형 맵에 랜덤하게 풀이 생성됩니다.
- 각 쥐토끼는 맵 위에 생성된 풀을 채집해서 풀무더기로 만듭니다.
  - 플레어이는 맵 위에 생성된 풀을 채집하려면 A를 세 번 눌러야합니다. 풀 채집이 완료되면 쥐토끼는 풀무더기를 입에 문 상태가 됩니다.
  - 입에 물 수 있는 풀무더기 숫자의 기본값은 1입니다.
  - 입에 풀무더기를 문 상태에서 S키를 누르면 입에 물고 있던 풀무더기를 맵에 놓습니다.
  - 풀무더기 위에서 S키를 누르면 풀무더기를 입에 뭅니다.
  - 풀무더기를 본인 굴로 가져가면 자동으로 풀무더기를 굴에 놓으며 풀무더기 누적 숫자가 +1 됩니다.
  - 풀무더기 위에서 혹은 풀무더기를 입에 문 상태에서 D키를 누르면 풀무더기를 먹습니다.
- 다른 쥐토끼 굴에 저장된 풀무더기를 S키로 누르면 풀무더기를 입에 뭅니다. 이때 해당 쥐토끼 굴에 저장된 풀무더기 누적 숫자는 -1이 됩니다. 
- 풀은 약초 혹은 독초입니다. 생김새는 다 같지만 색깔로 구분할 수 있습니다.
- 약초는 당장 먹으면 HP가 오릅니다. 시간이 지날수록 색깔이 어두워지며 HP 증가량이 감소합니다.
- 독초는 당장 먹으면 오히려 HP 깎입니다. 일정 시간 숙성시켜 독을 제거한 뒤 먹어서 HP를 올려봅시다. 숙성된 독초는 빛이 나며 일반 약초보다 HP증가량이 훨씬 높습니다.
- 랜덤하게 특별한 독초가 자라납니다.
  - 특별한 독초는 일정 시간 숙성 뒤 먹으면 랜덤하게 특수효과가 나타납니다.
    - 최대 HP가 20만큼 늘어납니다.
    - 이동속도가 2만큼 빨라집니다.
    - 한번에 입으로 물 수 있는 풀무더기 숫자가 2만큼 늘어납니다.
  - 특별한 독초는 숙성시키지 않고 먹으면 위의 특수효과가 반대로 나타납니다.
    - 최대 HP가 줄어듭니다. 단, 이미 최대 HP가 100인 경우 일반 독초와 같은 효과를 냅니다.
    - 이동속도가 느려집니다. 단, 이미 이동속도가 1인 경우 일반 독초와 같은 효과를 냅니다.
    - 한번에 입으로 물 수 있는 풀무더기 숫자가 1만큼 줄어듭니다. 단, 이미 숫자가 1인 경우 일반 독초와 같은 효과를 냅니다.
- 각 쥐토끼는 HP/최대HP를 가지고 있습니다.
  - 게임 시작 시 100/250으로 시작합니다.
  - 1초마다 HP가 1만큼 감소합니다
  - HP가 0인 경우 게임에서 패배합니다.
  - 상술한 약초와 독초 효과에 따라 HP 및 최대HP 수치는 변경됩니다.
- 각 쥐토끼의 이동속도 기본값은 1입니다. 특별한 독초의 효과에 따라 이동속도가 변경됩니다.
- 각 쥐토끼가 한번에 입으로 물 수 있는 풀무더기 숫자의 기본값은 1입니다. 특별한 독초의 효과에 따라 한번에 입으로 물 수 있는 풀무더기 숫자가 변경됩니다.
