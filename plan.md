# 게임 상세

쥐토끼를 조종하는 아케이드 게임을 html, css, js로 구현해야합니다. 다음을 참고해 구현해봅시다.

## 1. 게임 시작 및 타이틀

타이틀 화면에서 게임시작 버튼 누르고 시작합시다.

게임시작 버튼을 누르면 캐릭터 선택 창이 등장합니다.

## 2. 캐릭터 선택

본격적인 게임시작 전 캐릭터 선택 창에서 플레이 하고 싶은 쥐토끼를 선택합시다.

총 네 마리 중 한 마리 선택 가능하며, 1명의 플레이어가 다른 3명의 AI와 개인전으로 경쟁합니다.
- Pika: 그냥 쥐토끼
	- (HP/최대HP) = (150/250)으로 시작
	- 이동속도 2로 시작
	- 한번에 입으로 물 수 있는 풀더미 숫자 1로 시작
- Rpika: 알파카처럼 털이 복슬복슬한 쥐토끼
	- (HP/최대HP) = (200/300)으로 시작
	- 이동속도 1로 시작
	- 한번에 입으로 물 수 있는 풀더미 숫자 1로 시작
- Pyka: 매끈매끈한 쥐토끼
	- (HP/최대HP) = (100/250)으로 시작
	- 이동속도 4로 시작
	- 한번에 입으로 물 수 있는 풀더미 숫자 1로 시작
- Hika: 카우보이 쥐토끼
	- (HP/최대HP) = (100/250)으로 시작
	- 이동속도 2로 시작
	- 한번에 입으로 물 수 있는 풀더미 숫자 2로 시작

Enter/A/S/D키 중 아무 키를 눌러 쥐토끼를 선택합니다. 선택 후 스토리 화면으로 넘어갑니다.

## 3. 간단한 스토리 후 게임 규칙 / 조작 방법 제시

쥐토끼를 선택하면 간단한 스토리를 텍스트로 보여줍니다. 

플레이어가 원하면 다음 장면으로 바로 넘어갈 수 있게 SKIP 버튼도 만들어줍시다.

스토리가 끝나면 규칙 설명 및 조작방법 화면과 함께 게임시작 버튼이 나타납니다. 
  
게임시작 버튼을 눌러봅시다. 그러면 본 게임이 5초 카운트 후 시작합니다.

## 4. 게임 마무리 화면

승리/패배 조건이 달성되면 승리/패배 화면을 보여줍니다. 계속 버튼을 제시합니다.

게임오버 화면을 제시합니다. 승리 후 다시하기, 타이틀로, 캐릭터 재선택 버튼을 제시합니다.

## 5. 본 게임 상세

### A. 맵

기본 맵은 사각형 맵입니다. 

캐릭터 크기가 정사각형 한 칸이라고 가정할 때, 기본 맵 크기는 가로 50 칸 x 세로 50 칸입니다.

매 게임마다 장애물(바위/물)이 랜덤하게 배치됩니다. 장애물 위로는 지나갈 수 없습니다. 장애물은 각 쥐토끼 굴 반경 세 칸 이내에는 배치되지 않습니다.

사각형 맵 각 모서리에 각 쥐토끼의 굴이 있습니다. 각 쥐토끼의 굴은 매 게임마다 랜덤하게 배정됩니다. 
- 각 굴은 네 모서리에 정확히 고정시키고, 어느 쥐토끼가 어느 굴에 배치되는지만 랜덤 배정하면 됩니다.

모든 플레이어는 맵 전체를 볼 수 있습니다.

### B. 승리 / 패배 규칙

사각형 맵에 랜덤하게 풀(약초/독초/특수초)이 생성됩니다. 

각 쥐토끼는 맵 위에 생성된 풀을 수확해서 풀더미로 만듭니다.  
 
가장 먼저 자기 굴에 풀더미를 30개 쌓은 쥐토끼가 승리합니다.

플레이어가 먼저 30개를 쌓으면 승리 화면을 보여주고 게임을 마칩니다.

다른 AI가 먼저 30개를 쌓으면 패배 화면을 보여주고 게임을 마칩니다.

둘 이상의 캐릭터가 30개를 동시에 달성하면 플레이어가 승리합니다.

### C. 풀 정의

풀은 약초/독초/특수초 세 가지가 있습니다.
- 약초는 초록색입니다.
- 독초는 보라색입니다.
- 특수초는 노란색입니다.

(약초/독초/특수초 각각 생성 확률)

1초당 생성되는 약초의 개수는 Poisson(3) 분포를 따릅니다.

1초당 생성되는 독초의 개수는 Poisson(3) 분포를 따릅니다.

1초당 생성되는 특수초의 개수는 Poisson(0.5) 분포를 따릅니다.

맵에 생성된 풀의 개수가 20개를 넘기면 풀 생성을 잠시 중단하고 15개 아래로 내려가면 다시 생성을 합니다.

굴 반경 4칸 이내에는 풀을 생성하지 않습니다.

풀을 생성할 때는 오브젝트끼리 겹치지 않게 생성합니다.

풀은 풀더미가 되기 전까지는 운반할 수 없습니다.

풀이 풀더미가 되려면 캐릭터가 풀을 수확해야합니다. 

풀은 수확될 때까지 상태가 변하거나 사라지지 않습니다. 

### D. 풀더미 정의

풀더미는 풀과 다른 오브젝트입니다.

풀더미를 굴에 쌓아서 승리를 노리거나 소비해서 효과를 받을 수 있습니다. 

캐릭터에 관계없이 풀이 팔휘두르기를 세 번 맞으면 풀 수확이 완료되며 풀은 사라지고 풀더미가 생성됩니다. 
- 랜덤으로 풀더미가 1개 더 생성될 수 있습니다. 추가 생성되는 경우 기존 생성된 풀더미와 구분할 수 있게 3px 밑으로 겹치게 생성합니다.
- 풀더미가 추가로 생성될 확률은 Bernoulli(0.3)을 따릅니다.
- 쥐토끼가 풀더미를 입에 물지 않은 상태에서 수확을 완료하면 즉시 풀더미를 입에 문 상태가 됩니다.

풀더미는 약초더미/독초더미/특수초더미 세 가지가 있습니다.
- 약초더미는 초록색입니다.
- 독초더미는 보라색입니다.
- 특수초더미는 노란색입니다.

풀더미는 생성 즉시 숙성이 시작되고 10초가 지나면 숙성이 완료됩니다. 
- 숙성은 어디에서든 진행됩니다.
- 약초더미는 숙성이 완료되기 전 소비하면 HP가 증가합니다. 
	- HP 증가량은 Discrete Uniform(5, 15)를 따릅니다.
	- 숙성이 완료되면 하얀색으로 변하며 HP 증가량은 3으로 고정됩니다.
- 독초더미는 숙성이 완료되기 전 소비하면 HP가 감소합니다. 
	- HP 감소량은 Discrete Uniform(1, 10)을 따릅니다.
	- 숙성이 완료되면 약초더미로 변합니다.
- 특수초더미는 소비할 때 조건에 따라 랜덤하게 특수효과가 나타납니다. 
	- 숙성이 완료되면 빨간색 테두리가 반짝이는 효과가 발생합니다. 숙성이 완료된 상태에서 소비할 때 다음 세 효과 중 하나가 나타납니다. 각 효과가 나타날 확률은 균등합니다.
		- HP와 최대 HP가 20만큼 늘어납니다.
	    - 이동속도가 1만큼 빨라집니다.
	    - 한번에 입으로 물 수 있는 풀더미 숫자가 2만큼 늘어납니다.
	- 숙성하지 않고 소비할 때 다음 세 효과 중 하나가 나타납니다. 각 효과가 나타날 확률은 균등합니다.
	    - 최대 HP가 20만큼 줄어듭니다. 단, 최대 HP를 100보다 감소시키진 않습니다. 이미 최대 HP가 100인 경우 일반 독초더미와 같은 효과를 냅니다. 
	    - 이동속도가 1만큼 느려집니다. 단, 이미 이동속도가 1인 경우 일반 독초더미와 같은 효과를 냅니다.
	    - 한번에 입으로 물 수 있는 풀더미 숫자가 1만큼 줄어듭니다. 단, 이미 1인 경우 일반 독초더미와 같은 효과를 냅니다.

풀더미를 본인 굴로 가져가면 자동으로 풀더미를 굴에 놓으며 풀더미 누적 숫자가 +1 됩니다.

본인 혹은 다른 쥐토끼 굴에 저장된 풀더미를 S키로 누르면 풀더미를 입에 뭅니다. 이때 해당 쥐토끼 굴에 저장된 풀더미 누적 숫자는 -1이 됩니다. 

본인 혹은 다른 쥐토끼 굴에 저장된 풀더미를 D키로 누르면 풀더미를 소비합니다. 이때 해당 쥐토끼 굴에 저장된 풀더미 누적 숫자는 -1이 됩니다. 

풀더미를 굴에 쌓은 경우 가장 마지막에 쌓인 풀더미만 아이콘 표시를 합니다.

### E. 조작 방법

조작 방법은 다음과 같습니다.

이동 키
- 상하좌우 방향키
- 각 캐릭터는 정사각형 한 칸의 크기입니다. 

A: 앞다리 휘두르기 
- A키를 누를 때마다 앞다리를 휘두르는 모션을 취합니다.
- 캐릭터와 풀 간 거리가 1px이내일 때 풀을 바라본 상태에서 앞다리를 세 번 휘두르면 풀을 풀더미로 변환시킵니다.

S: 풀더미 입에 물기 / 놓기
- 캐릭터가 풀더미 반경 1px 이내에 있고, 풀더미를 입에 물지 않은 상태에서 S키를 누르면 풀더미를 입에 문 상태가 됩니다. 
- 캐릭터가 풀더미를 입에 문 상태에서 S키를 누르면 입에 물고 있던 풀더미를 맵에 놓습니다.
	- 풀 위에 풀더미가 같이 있는 경우 S는 풀더미를 우선합니다.
	- 풀더미를 놓는 경우 캐릭터 밑에 겹쳐진 상태로 놓습니다.
	- 풀더미 위에 또다른 풀더미를 놓는 경우 겹치지 않게 배치됩니다.

D: 풀더미 소비
- 풀더미 반경 1px이내 혹은 풀더미를 입에 문 상태에서 D키를 누르면 풀더미를 소비합니다.

A/S/D는 이동 중에도 조작 가능합니다.

### F. 캐릭터 상세

각 쥐토끼는 1초마다 HP가 1만큼 감소합니다

HP가 0인 경우 게임에서 패배합니다.
- 플레이어 HP가 0이 되면 패배 화면을 보여주고 게임을 마칩니다.
- 플레이어가 아닌 캐릭터의 HP가 0이 되면 해당 캐릭터는 동작을 멈추고 드러눕는 그림으로 바뀝니다. 다른 캐릭터의 게임은 계속 진행됩니다.
- HP는 최대 HP보다 항상 작거나 같습니다.
